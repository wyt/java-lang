###设计模式

* `samples.callback` 回调模式(异步示例)
  * `CallBack` 回调接口
  * `Customer` 客户
  * `CustomerService` 客服中心
  * `Test` 测试入口
* `samples.callback2` 回调模式(异步示例)，lambda方式替换实现回调接口
  * `CallBack` 回调接口
  * `Customer` 客户
  * `CustomerService` 客服中心
  * `Test` 测试入口
* `samples.callback` 监听器模式(同步示例)
  * `FinishedListener` 监听器接口
  * `Customer` 客户
  * `CustomerService` 客服中心
  * `Test` 测试入口
